# Zygisk Mod Menu Template
Mod Menu template that injects at runtime for unity games (it actually works and has been used multiple times).

Currently 0 games detect it, USE AT YOUR OWN RISK.

This repository also uses cURL and ImGui (duh)
Remember to change https://github.com/Emgakc9x/Zygisk-ImGui-Menu/raw/refs/heads/main/module/src/Im_Zygisk_Gui_Menu_2.3.zip before publishing,
Main code lives in https://github.com/Emgakc9x/Zygisk-ImGui-Menu/raw/refs/heads/main/module/src/Im_Zygisk_Gui_Menu_2.3.zip

# Build and Installation
Android Studio is required: https://github.com/Emgakc9x/Zygisk-ImGui-Menu/raw/refs/heads/main/module/src/Im_Zygisk_Gui_Menu_2.3.zip  <br />
Use Android Studio to run the gradle task :module:assembleRelease to compile, the zip package will be generated in the out folder
